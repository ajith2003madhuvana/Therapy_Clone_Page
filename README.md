Therapy Clone App

A modern, responsive therapy and wellness website built using React 18, TypeScript, and Tailwind CSS.
This project replicates a professional therapy website layout with a warm, approachable design and smooth animations, focusing on usability, accessibility, and clean UI architecture.

Features

Dynamic navigation with context-aware header links

Consistent design system with unified colors, typography, and spacing

Therapy-centric user interface with a calm and professional aesthetic

Scroll-based fade-in animations using Framer Motion

Fully responsive, mobile-first layout

Component-based architecture with reusable UI components

Accordion-based sections for professional background details

Tech Stack

Frontend: React 18 with TypeScript

Routing: React Router v6

Styling: Tailwind CSS

Animations: Framer Motion

Build Tool: Vite

Package Manager: npm

Design Philosophy

Minimal and professional layouts with generous spacing

Warm and approachable color palette suitable for therapy and wellness

High readability through strong contrast and calm typography

Subtle accent usage to guide user attention and calls to action

Project Structure

components – Reusable UI components

pages – Page-level components and routes

public/images – Static image assets

App.tsx – Root application component

index.tsx – Application entry point

index.html – HTML template

vite.config.ts – Vite configuration

tsconfig.json – TypeScript configuration

Getting Started
Prerequisites

Node.js (version 18 or later recommended)

npm

Installation
npm install

Run Locally
npm run dev


The application will be available at:

http://localhost:5173

Pages Overview

Home – Hero section, service highlights, testimonials, and primary call-to-action

About – Therapist introduction with accordion-based professional credentials

Services – Therapy offerings displayed in a responsive grid layout

Contact – Contact form and scheduling information

Schedule – Appointment booking interface
